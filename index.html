✦ Edit Resources

Resources, environments, and categories are managed through the admin UI. Run the following command in your terminal, then open the link:

npm run admin
Open Admin UI → http://localhost:3001

The admin UI lets you add, edit, and delete resources, environments, envSets, and categories. Changes are validated and rebuilt automatically.
